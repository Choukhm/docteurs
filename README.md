# Docteurs

Projet du Vendredi 26 Juillet THP

## Versions

Ruby version : 2.5.1 
Rails version : 5.2.0

## Mode d'emploi
Récupère le repo Github (git clone ou zip) <br/>
Utilises le terminal pour te placer dans le dossier récupéré, ensuite tapes :<br/>
<br/>

* `bundle install`<br/>

* `rails db:migrate`<br/>

* `rails db:seed`
